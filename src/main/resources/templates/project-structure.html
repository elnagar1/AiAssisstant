Automation Project Structure

This is a typical structure for a Java Selenium (TestNG, POM) automation project.
Add the generated Test class and Page class in the highlighted folders below:

  • automation-project-root/
    • src/
      • main/
        • java/
          • com/
            • yourcompany/
              • pages/ ← Place Page class here
                • GeneratedPage.java
              • utils/
      • test/
        • java/
          • com/
            • yourcompany/
              • tests/ ← Place Test class here
                • GeneratedTest.java
    • drivers/
    • pom.xml
    • testng.xml
    • README.md
Note:
- Test class (e.g., GeneratedTest.java) goes in src/test/java/com/yourcompany/tests/.
- Page class (e.g., GeneratedPage.java) goes in src/main/java/com/yourcompany/pages/.
- Adjust yourcompany to match your actual package structure.
- GeneratedTest.java & GeneratedPage.java: These are the generated automation classes.
